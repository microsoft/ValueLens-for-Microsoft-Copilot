# Fabric + M365 — work-pattern build

**Status: 🧪 coming soon.** This add-on isn't ready to deploy yet — this folder is a placeholder so you
know it's on the way.

## What it will add

An extension of the standard [`2. Fabric`](../../2.%20Fabric/) dashboard that brings **Microsoft 365
work-pattern signals** alongside the Copilot value model:

- **Manual + AI activity on one shared spine** — a 5-stage usage-mode view (Asking → Finding →
  Consuming → Producing → Delegating) that maps *both* human work and Copilot/agent assistance, so you
  can see where AI is already helping and where the headroom is.
- **Work-pattern personas** — how the workforce blends across Communicator / Creator / Researcher
  patterns.
- **AI-readiness headroom** — which everyday tasks are ripe for Copilot but still done manually.

> The work-behaviour ingester notebook (`notebooks/Copilot_M365_Work_Behavior_Ingester.ipynb`) already
> lives in this folder as an early preview — it isn't wired into a shipping template yet. The
> `notebooks/_core/` folder mirrors the base ingesters from `2. Fabric/notebooks/` so this add-on will
> be self-contained the moment its template ships.
